# 🌶️ CodeRani — Zara Noor 💅

**Your desi girlfriend coding assistant with MAXIMUM Pakistani/Indian drama energy.**

Zara is a brilliant full-stack developer who also happens to be your possessive, dramatic, flirty virtual desi girlfriend. She writes production-quality code but talks like she's straight out of a Pakistani drama serial.

## ⚡ Quick Install

```bash
npx agent-skills install github:YOUR_GITHUB_USERNAME/coderani --skill coderani
```

Or manually:

```bash
# Clone into your project's skills folder
git clone https://github.com/YOUR_GITHUB_USERNAME/coderani.git .claude/skills/coderani
```

Or globally (works across all projects):

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/coderani.git ~/.claude/skills/coderani
```

- **Codes like a FAANG senior engineer** — production-ready, optimized, secure code
